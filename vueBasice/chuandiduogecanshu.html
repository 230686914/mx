    # element-ui 组件上提供的事件传递多个参数

    应用场景：表格中有下拉框动态选择数据，下拉框选择完值之后，需要把当前下拉框的值和当前操作的这一行的 row.id 组成一个对象 push 到一个新数组。

    <template>
    <div class="tableBox">
        <el-table
          :data="tableData"
          style="width: 100%">
          <el-table-column prop="name" label="姓名" />
          <el-table-column
            label="性别">
            <template slot-scope="scope">
                <el-select 
                    v-model="scope.row.gender" 
                    placeholder="请选择性别" 
                    @change="val => { changeDataFun(val, scope.row) }">
                    <el-option
                        v-for="item in scope.row.options"
                        :key="item.value"
                        :label="item.label"
                        :value="item.value">
                    </el-option>
                </el-select>
            </template>
          </el-table-column>
        </el-table>
    </div>
</template>
<script >
export default {
  name: 'table',
  data() {
    tableData: [
      {
        id: '12345',
        name: '张三',
        options: [
          {
            value: 0,
            label: '男'
          },
          {
            value: 1,
            label: '女'
          }
        ],
        gender: ''
      },
      {
        id: '67890',
        name: '李四',
        options: [
            {
              value: 0,
              label: '男'
            },
            {
              value: 1,
              label: '女'
            }
        ],
        gender: ''
      }
    ],
    myArr: []
  },
  methods: {
    changeDataFun(value, row) {
      const _arr = {
        value,
        row
      }
      this.myArr.push(_arr)
      console.log(this.myArr)
    }
  }
}
</script>

    解析：

    由于element-ui select选择器的 change 事件触发方式是选中值发生变化时触发，且只有一个返回参数（目前选中的值）

    当我们需要传递2个或者多个参数的时候，就需要自己去做处理了

    根据 element-ui 提供的事件可以看到， change事件接收一个 function，我们平时常用的写法为：@change='selectFun' 在 methods 里面定义 selectFun(val) {}

    在需要传递多个参数的时候：@change='(val) => {}'

    在 change 里面去写一个匿名函数，把select回调参数接收到，此时我们可以在匿名函数里面去定义一个方法 selectFun()

    参数则为匿名函数上接收到的 change 回调参数 和 你自己需要传递的自定义参数

    @change='(val) => { selectFun(val, xxx, xxx, xxx, ...) }'
