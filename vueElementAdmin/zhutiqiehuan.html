    # 主题切换

    基于 vue-element-ui 实现项目换肤功能

    思路：要设置3套不同的主题 - theme / theme2 / theme3

    切换不同的主题时在body上添加主题对应的class类

    所有涉及到主题切换的模块样式都写在 主题对应的 class 类下面

    在 styles 文件下面新建文件夹 - topicSwitch

topicSwitch => 新建3个文件分别为：theme.scss / theme1.scss / theme2.scss

在 utils 文件夹下面的 index,js 里面写个公共类并导出  用于 主题切换
